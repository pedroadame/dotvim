## Vim configuration

### Installation
#### Clone it
```
git clone --recursive https://github.com/pedroadame/dotvim ~/dotvim
ln -s ~/dotvim ~/.vim
```
#### Configure it
```
vim ~/.vim/vimrc
```
Comment unneeded settings and add/remove plugins

#### Install plugins
1. Run vim
2. Run :PluginInstall
3. Restart vim and you're ready to go!

#### Update repo
```
git pull --rebase
```
